How to Add

チームの追加・変更

新しいチームページは次の4ステップで追加します。詳細な手順はリポジトリの CONTENT-GUIDE.md を参照してください。

  1. teams/_template.html を複製して teams/チーム名ローマ字.html を作ります。
  2. 複製したファイルの <title> と page-hero のチーム名を書き換えます。
  3. js/ops-config.js の teams 配列に1行追加します。
  4. pushすれば、ナビ・この一覧・ハブトップに自動反映されます。
